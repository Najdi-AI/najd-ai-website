# Najd AI Solutions — Component Library

The 47 [Aceternity UI](https://ui.aceternity.com) components used to build this site, vendored
with their **full source** and a **README** each. Every component has been recoloured to the Najd
brand palette (green `#2bb673` · blue `#2699D6` · teal `#215877` · navy `#0d2745`).

Each folder contains the component source (`<name>.tsx`), its demo, and `README.md` (what it is,
dependencies, props, usage, and where it's used on the site). Install any component standalone with:

```bash
npx shadcn@latest add @aceternity/<demo-name>
```

Browse them live at **`/components`** on the site.
